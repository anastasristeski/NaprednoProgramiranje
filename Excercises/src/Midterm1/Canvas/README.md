# ShapesApplication

The `ShapesApplication` class is designed to manage and process data for multiple canvases that display geometric shapes in the form of squares.

## Class Definition

### Constructor
- **`ShapesApplication()`**  
  Initializes an instance of the `ShapesApplication` class.
